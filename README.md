## What is DSL SLAM?
* DSL SLAM is a Distributed SOLiD-based LiDAR SLAM, which is a modified version of [DiSCO-SLAM](https://github.com/RobustFieldAutonomyLab/DiSCo-SLAM). 

## Dependencies
* Ubuntu 20.04
* GTSAM (Develop version)
* SOLiD
  * libnabo 1.0.7
